# Random Welcome Angular Project

This Angular project displays a random welcome message each time the application is loaded. It is designed for easy expansion and modularity, allowing for the addition of new features and components as needed.

## Project Structure

- **src/**: Contains the source code for the application.
  - **app/**: The main application module and components.
  - **core/**: Core services, interceptors, and guards.
  - **shared/**: Shared components and utilities.
  - **features/**: Feature modules for different parts of the application.
  - **environments/**: Environment-specific configurations.
  - **assets/**: Static assets for the application.
  
## Getting Started

1. **Clone the repository**:
   ```
   git clone <repository-url>
   cd random-welcome-angular
   ```

2. **Install dependencies**:
   ```
   npm install
   ```

3. **Run the application**:
   ```
   ng serve
   ```

4. **Open your browser**:
   Navigate to `http://localhost:4200` to see the application in action.

## Features

- Displays a random welcome message on each load.
- Modular architecture for easy maintenance and scalability.
- Configured with Angular best practices.

## Future Enhancements
